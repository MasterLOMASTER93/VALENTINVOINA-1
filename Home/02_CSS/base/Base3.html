Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью,как показано в данном примере

Статья1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "Статьях" сделайте отступ снизу от соседей равным.2em, при помощи свойства margin-bottom

Статья2

Размер блока:
Занимает всю доступную ширину
Дролжен присутствовать цвет фона
Во всех "Статьях" сделайте отступ снизу от соседей раным .2em, при помощи свойства margin-botton.Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок,подвал.Занимает всю доступную ширину.Высота определяется помещенным внутрь контентом
